# مشروع وكيل الذكاء الاصطناعي الشامل للبرمجة والتصميم

## نظرة عامة على المشروع

هذا المشروع يهدف إلى تطوير وكيل ذكاء اصطناعي شامل يتعلم ذاتياً في مجالي البرمجة والتصميم. الهدف النهائي هو بناء أداة إنتاجية متكاملة يمكنها إدارة المشاريع، أتمتة المهام المتكررة، وتحليل البيانات وتقديم التقارير. يعتمد الوكيل على التعلم المستمر من المصادر المفتوحة، ويستخدم آليات تقييم وتحسين ذاتي لتعزيز قدراته.

## التقنيات المستخدمة

تم اختيار التقنيات التالية لبناء هذا المشروع لشموليتها ومرونتها:

*   **الواجهة الأمامية (Frontend):** React.js
*   **الواجهة الخلفية (Backend):** Node.js مع Express.js
*   **قاعدة البيانات (Database):** PostgreSQL
*   **أتمتة سير العمل (Workflow Automation):** n8n

## بنية الوكيل

يتكون الوكيل من ست وحدات رئيسية تعمل بتناغم:

1.  **وحدة جمع البيانات:** لجمع المعلومات من مصادر مفتوحة.
2.  **وحدة معالجة البيانات:** لتنظيف وتحليل البيانات المجمعة.
3.  **وحدة التعلم:** لاكتساب المعرفة وتحديث النماذج المعرفية.
4.  **وحدة التخطيط:** لتحليل المهام وإنشاء خطط العمل.
5.  **وحدة التنفيذ:** لكتابة الكود وتصميم الواجهات.
6.  **وحدة التقييم:** لقياس الأداء وتقديم الملاحظات.

يمكن العثور على توثيق مفصل لبنية الوكيل في ملف `agent_architecture.md`.

## إعداد وتشغيل المشروع

لإعداد وتشغيل هذا المشروع، اتبع الخطوات التالية:

### 1. استنساخ المستودع

```bash
git clone <رابط المستودع الخاص بك>
cd <اسم مجلد المشروع>
```

### 2. إعداد الواجهة الأمامية (React.js)

انتقل إلى مجلد الواجهة الأمامية وقم بتثبيت التبعيات وتشغيل خادم التطوير:

```bash
cd ai-agent-environment
pnpm install
pnpm run dev --host
```

سيتم تشغيل الواجهة الأمامية على `http://localhost:5173` (أو منفذ آخر متاح).

### 3. إعداد الواجهة الخلفية (Flask)

افتح نافذة طرفية جديدة، ثم انتقل إلى مجلد الواجهة الخلفية وقم بإنشاء البيئة الافتراضية، تثبيت التبعيات، وتشغيل الخادم:

```bash
cd ai-agent-backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python src/main.py
```

سيتم تشغيل الواجهة الخلفية على `http://localhost:5000`.

### 4. الوصول إلى واجهة الوكيل

بعد تشغيل كل من الواجهة الأمامية والخلفية، يمكنك الوصول إلى واجهة بيئة تعلم وكيل الذكاء الاصطناعي الشامل عبر المتصفح على العنوان الذي تم تشغيل الواجهة الأمامية عليه (عادةً `http://localhost:5173`).

### 5. النشر الدائم (Permanent Deployment)

لتحويل هذا المشروع إلى موقع ويب دائم، يمكنك استخدام منصات الاستضافة السحابية التالية. تم تجهيز المشروع بملفات التكوين اللازمة للنشر السريع.
